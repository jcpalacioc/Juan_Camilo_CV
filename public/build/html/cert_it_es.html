Google Data Analytics Certificate

2023-11-13

Verificar

Este curso trata sobre cómo convertirse en un analista de datos. Se centra en cómo generar información muy útil para las partes interesadas con marcos y procesos bien definidos de los datos, utilizando herramientas como BigQuery, Tableau, hojas de cálculo y R para limpiar, filtrar y crear gráficos e información sobre los datos. Hace mucho enfasis en las habilidades comunicativas y trabajo en equipo con los especialistas de negocio.

Certificado Full-Stack Web

2023-07-28

Verificar

Este curso trata sobre cómo convertirse en un desarrollador web full-stack, comenzando con lenguajes front-end como HTML, CSS y JavaScript, y luego a usar SASS y Gulp para automatizar tareas y desarrollar rápidamente.
